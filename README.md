# GameProject – Clean Export

이 폴더는 **WoodLand3D** 프로젝트의 스크립트·씬·문서·설정만 담은 내보내기입니다.  
전체 유니티 에셋(메시, 텍스처, 오디오 등)은 포함하지 않습니다.

- **저장소**: https://github.com/akj2013/GameProject
- **업데이트 방법**: WoodLand3D 루트에서 `.\Export-Clean.ps1` 실행 후, 이 폴더에서 `git add` → `commit` → `push`
- **자세한 워크플로**: Assets/Docs/15_GIT_WORKFLOW.md 참고
