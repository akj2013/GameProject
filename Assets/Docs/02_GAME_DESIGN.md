# Game Design

## 장르

자원 채집 + 마을 확장 + 캐릭터 조작 게임

## 플레이 방식

플레이어는 캐릭터를 직접 이동시키며 자원을 채집한다.

채집 대상:

- 나무
- 돌
- 광석
- 농작물

## 카메라

아이소메트릭 스타일

Rotation
X: 45°
Y: 45°

넓은 시야로 마을과 풍경을 보여주는 것이 목표.

## 맵 구조

사각 타일 기반

각 타일은 다음 상태를 가진다:

- Locked (구름으로 가려짐)
- Unlocked (사용 가능)

타일 확장은 자원 또는 금화를 사용한다.

## 지역 유형

- 숲 지역
- 광산 지역
- 농장 지역
- 마을 지역
- 시장
- 사찰
