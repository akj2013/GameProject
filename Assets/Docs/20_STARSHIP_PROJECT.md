# 1. MVP 기능 목록

## MVP 목표

플레이어가 아래 1사이클을 실제로 경험할 수 있으면 성공.

**정박지 → 출항 → 항해 → 채집 포인트 발견 → 채집 → 자원 획득 → 회항 → 판매/강화 → 재출항**

즉, 이 한 바퀴만 돌면 된다.

---

## A. 필수 기능

### 1) 정박지 화면

* 비행선이 하늘섬 옆에 떠 있는 상태
* 버튼 3개만 존재

  * 출항
  * 판매
  * 강화

### 2) 출항 전환

* 버튼 누르면 짧은 전환 연출
* 카메라 이동 + 비행선 위치 변경
* 항해 씬 상태로 전환

### 3) 항해 화면

* 비행선 표시
* 좌우 이동 가능
* 전진은 실제 이동이 아니라 연출

  * 구름 스크롤
  * 바닥 풍경 스크롤
  * 엔진 이펙트
  * 카메라 약한 흔들림

### 4) 채집 포인트 생성

* 화면 내 일정 위치에 채집 포인트 출현
* 불빛/작은 발광으로 표시
* 가까이 가면 패널 UI 등장

### 5) 채집 포인트 UI

* 자원 종류
* 획득량
* 채집 시간
* 채집 버튼

### 6) 연결선 생성

* 채집 버튼 누르면 비행선과 포인트를 잇는 빛나는 선 생성
* 최대 1개만 연결
* 일정 거리 이상 멀어지면 끊김

### 7) 채집 완료 처리

* 일정 시간 후 자원 지급
* 상단 메시지 표시

  * 예: 일반 목재 15개를 채집하였습니다
* 비행선 갑판에 적재 오브젝트 1개 추가

### 8) 회항

* 일정 시간 후 자동 회항 또는 회항 버튼
* 정박지로 복귀 연출

### 9) 판매

* 정박지에서 현재 적재/보유 자원을 골드로 전환

### 10) 강화

* 간단한 업그레이드 1~2개

  * 이동속도
  * 적재량 또는 채집속도

## B. MVP에서 꼭 넣지 말 것

이건 나중이다.

* 여러 월드맵
* 여러 정박지 섬 기능 분리
* 수리 시스템
* 연구 시간 단축
* 캐쉬 상점
* 부품별 상세 세팅
* 연료 종류
* 레이저 종류 다변화
* 동시 연결선 2개 이상
* 복잡한 적재 중량 패널티
* 국가별 맵 테마
* 구름 구멍 정교한 마스킹 시스템
* 화물 종류 5개 이상

## C. MVP 자원 종류

처음엔 2종이면 충분하다.

* 목재
* 광석

골드는 판매용 통화 1종.

---

## D. MVP 업그레이드 종류

* **Engine Level**

* **Cargo Level**

  * 1회 항해에서 획득 가능한 최대 자원량 증가

채집속도 업그레이드는 3번째 정도로 추가 가능.

# 다음에 만들어야 할 것 우선순위

## 1. 정박지 하늘섬

가장 먼저.

왜냐면 네 게임은

* 출항 전
* 회항 후
* 판매 / 강화 / 상점 / 연구

이 전부가 정박지에서 일어나니까,
정박지가 없으면 게임 루프의 절반이 비어 보여.

### 필요한 것

* 정박지 하늘섬 메인 비주얼 1장
* 정박 상태 비행선이 옆에 떠 있을 때 어울리는 구도
* 섬 위 건물은 너무 많지 말고 기능별 포인트가 읽히게

### 처음엔 이렇게 단순하게

* 중앙 정박탑 / 계류장
* 상점 건물 1개
* 연구소 느낌 건물 1개
* 조명 / 등불 / 푸른 수정 같은 포인트

**핵심은 정박지가 “베이스캠프”처럼 보여야 한다는 것**이야.

---

## 2. 항해 중 바닥 풍경 스프라이트

두 번째.

네 게임은 실제로 넓은 3D 월드를 안 만들고,
**아래에 깔리는 2D 풍경**으로 이동감을 만들 거잖아.

그래서 항해 중 아래에 보이는 풍경이 필요해.

### 우선 필요한 종류

* 숲 지역 바닥 풍경 1장
* 강이 흐르는 풍경 1장
* 바위/광산 느낌 풍경 1장

### 포인트

* 너무 디테일하면 오히려 반복 티 남
* 구름으로 일부 가려질 걸 전제로 만들어야 함
* 전체가 “지도”처럼 보이면 안 되고, 아래 멀리 내려다보는 느낌이어야 함

---

## 3. 채집 포인트 비주얼

세 번째. 이것도 매우 중요.

이 게임의 핵심 재미가
**“구름 사이에서 채집 포인트 발견”** 이니까
그 포인트가 예뻐야 한다.

### 먼저 필요한 종류

* 벌목 포인트
* 광석 포인트
* 희귀 수정 포인트 정도

### 형태

완전 복잡한 건 필요 없고,
구름 사이로 보이는 **빛나는 작은 구조물/작업장/광맥 입구** 같은 느낌이면 된다.

예:

* 벌목 포인트: 작은 작업 오두막 + 통나무 흔적 + 따뜻한 주황빛
* 광석 포인트: 수정 광맥 + 푸른빛
* 희귀 포인트: 신비한 유적/별빛 샘

---

## 4. 연결선 이펙트 디자인 기준

이건 이미지라기보다 **비주얼 스타일 가이드**를 먼저 정해야 해.

왜 중요하냐면 연결선은 네 게임의 시그니처라서 그래.

### 정해야 할 것

* 선 색
* 중심광 / 외곽광
* 흔들림 정도
* 채집 중 끝점 파편 효과
* 벌목/광석/수정마다 선 색이 다를지

처음엔 실제 VFX 제작보다
**컨셉 이미지 1장** 정도로 “이런 느낌”을 잡아두는 게 좋다.

---

## 5. UI 스타일 시안

지금부터 필요해.

왜냐면 네 게임은

* 채집 패널
* 자원 바
* 출항 버튼
* 판매/강화 버튼
* 정보 팝업

이런 UI가 꽤 많이 보이는데,
UI 스타일이 배경과 안 맞으면 갑자기 싸구려 모바일 게임처럼 보일 수 있어.

### 먼저 잡을 것

* 상단 자원 바
* 채집 포인트 패널
* 정박지 버튼 3종
* 정보 팝업 스타일

스타일은 추천이
**황동 + 짙은 남색 + 푸른 발광 + 따뜻한 등불색** 조합이 좋아.

---

## 6. 비행선 적재물 비주얼

이건 그 다음.

채집 완료 후 갑판에 쌓이는 게 보여야 보상감이 생겨.

### 처음 필요한 것

* 통나무 묶음
* 광석 상자 또는 광물 덩어리
* 수정 컨테이너 같은 것

이건 고퀄 복잡 모델까진 아니고,
비행선 갑판에 올렸을 때 “쌓였다”는 인상이 나면 충분해.

나중이지만 중요.

이미 네가 말했듯이 달은 따로 만들 예정이니까,
스카이박스에 얹을지, 월드 오브젝트로 띄울지 정해야 해.

추천은 **배경 오브젝트로 따로** 두는 거야.
그래야 위치, 크기, 발광을 조절하기 쉽다.
