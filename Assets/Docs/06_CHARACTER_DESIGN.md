# Character Design

## 캐릭터 스타일

Stylized Chibi Character

특징:

- 큰 머리
- 작은 몸
- 귀여운 얼굴
- Low Poly 스타일

## 제작 방식

Meshy AI를 사용하여 생성

Base Character + Outfit 방식으로 제작

## Base Character

모든 NPC는 동일한 기본 캐릭터 모델을 사용한다.

## Outfit 시스템

직업은 옷과 장비로 구분한다.

예:

농부
- 삿갓
- 농부 옷

광부
- 헬멧
- 곡괭이

관리
- 관모
- 관복

왕
- 왕관
- 왕복

## 장점

- 애니메이션 공유 가능
- 제작 속도 빠름
- 캐릭터 통일성 유지

## 추가 메모

- 캐릭터 실루엣은 멀리서 봐도 직업이 구분되도록, 모자/도구/색상 중 하나는 강하게 차이를 준다.
- 장비/의상은 가능하면 텍스처 색 변경과 소수의 메쉬 교체만으로 버전업할 수 있게 설계해, 나중에 스킨 작업 부담을 줄인다.
- 플레이어 캐릭터는 감정 표현(이모티콘, 제스처) 위주로, NPC는 반복 애니메이션 루프 위주로 리소스를 배분한다.
