# 물 타일(TileHex_25) 플레이어 차단 + 다리로만 통과

목표: **물 타일**에는 플레이어가 직접 들어가지 못하고, **다리/나무 기둥** 위로만 건널 수 있게 하기.

---

## 1. 요약

- **물 타일(TileHex_25)** → **Water** 레이어 + 콜라이더 유지 → 플레이어와 **충돌**해서 접근 차단.
- **다리/나무 기둥** → 물 타일 **위**에 배치하고, **별도 콜라이더**로 올라갈 수 있게 → 그 위로만 건너감.

프로젝트에 이미 **Water**, **Player** 레이어가 있으므로, 타일 레이어만 바꾸고 충돌 매트릭스만 확인하면 됩니다.

---

## 2. 할 일

### 2-1. 물 타일(TileHex_25) 레이어

1. **TileHex_25** 프리팹 열기  
   `Assets/Pandazole_Ultimate_Pack/.../Prefabs/TileHex_25.prefab` 더블클릭.
2. 루트 오브젝트 선택 → Inspector **Layer** → **Water** 선택.
3. 프리팹 저장.

(씬에 이미 놓인 TileHex_25 인스턴스는 프리팹 적용 시 같이 바뀌거나, 인스턴스만 선택해서 Layer를 Water로 바꿔도 됨.)

### 2-2. 충돌 매트릭스 확인

- **Edit → Project Settings → Physics** → **Layer Collision Matrix**.
- **Player**와 **Water**가 **충돌하는지(체크)** 확인.  
  - 체크되어 있으면 → 물 타일 콜라이더와 플레이어가 부딪혀서 **물로 직접 접근 불가**.
  - 체크 해제되어 있으면 → 체크해서 플레이어가 물과 충돌하도록 설정.

### 2-3. 다리/나무 기둥으로 건너가게 하기

- **다리**나 **나무 기둥**은 **별도 오브젝트**로 두고, **물 타일 위**(같은 타일 위치, Y만 더 높게)에 배치.
- 이 오브젝트에는 **Collider**(BoxCollider / MeshCollider 등)를 달아서 **발로 디딜 수 있는 면**을 만듦.
- 레이어는 **Default** 또는 **Ground** 등 **플레이어와 충돌하는 레이어**로 두면 됨.

동작 방식:

- 플레이어가 **물만 밟으려고** 하면 → 물 타일(Water) 콜라이더와 충돌 → 막힘.
- **다리 위**로 올라가면 → 다리 콜라이더와 충돌해서 그 위에 서 있음 → 다리는 물 타일보다 **Y가 높게** 있으므로, 물 콜라이더는 **발 아래**에 있음.
- 물 타일 콜라이더가 **타일 표면만** 덮는 두께라면, 다리를 조금만 올려도 플레이어는 다리만 밟고 물 콜라이더와 겹치지 않음.

**주의:**  
물 타일 메쉬가 두꺼우면 MeshCollider도 두꺼워질 수 있음. 그러면 다리를 **충분히 높게** 배치해서, 플레이어가 다리 위에 설 때 **발이 물 콜라이더와 겹치지 않도록** 해야 함. 필요하면 물 타일 메쉬를 얇게 쓰거나, 콜라이더만 별도로 얇은 BoxCollider로 두는 방법도 있음.

---

## 3. 정리

| 대상 | 레이어 | 콜라이더 | 결과 |
|------|--------|----------|------|
| TileHex_25 (물) | **Water** | 유지 (MeshCollider) | 플레이어와 충돌 → 물로 직접 접근 불가 |
| 다리 / 나무 기둥 | Default 등 | 있음 (발로 디딜 면) | 물 타일 **위**에 배치 → 그 위로만 건널 수 있음 |

레이어만 Water로 바꾸고, 다리는 물 타일보다 **위**에 두면 “물은 막고, 다리로만 건너기”가 됩니다.

---

## 4. 플레이어가 물을 그냥 지나갈 때 (해결)

**원인:** 플레이어 이동에 쓰는 **CharacterLocomotion** 스크립트의 `Start()`에서  
`characterController.detectCollisions = false` 로 되어 있으면, CharacterController가 **모든 콜라이더와 충돌을 하지 않아** 물 타일을 통과합니다.

**조치:** `CharacterLocomotion.cs`에서 `detectCollisions = true` 로 바꿔 두었습니다.  
이제 Player 레이어와 Water 레이어가 충돌하도록 되어 있으면, 물 타일에서 막힙니다.
