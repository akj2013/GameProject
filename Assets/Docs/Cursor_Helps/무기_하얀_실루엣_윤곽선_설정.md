# 무기 하얀 실루엣(윤곽선) 표현

`PP_Theme_01_Axe_Two_Handed_002 (1)` 처럼 씬에 추가한 무기를 **하얀색 실루엣(윤곽선)** 으로 보이게 하는 방법입니다.

## 1. 전용 머티리얼 사용 (권장)

프로젝트에 **PP_Weapon_WhiteSilhouette** 머티리얼이 있습니다.

1. **Project** 창에서  
   `Assets/PurePoly/Free Fantasy RPG Weapons/Materials/PP_Weapon_WhiteSilhouette` 선택
2. 씬의 **PP_Theme_01_Axe_Two_Handed_002 (1)** 오브젝트 선택
3. **Inspector**에서 **Mesh Renderer** > **Materials** > **Element 0**  
   → 현재 머티리얼 슬롯에 **PP_Weapon_WhiteSilhouette** 을 드래그해서 넣기  
   (또는 원형 버튼으로 해당 머티리얼 선택)

이 머티리얼은 **URP Unlit** 셰이더 + **흰색(_BaseColor)** 이라서, 조명 없이 무기 형태가 하얀 실루엣으로 보입니다.

## 2. 직접 머티리얼 만들기

1. **Project** 창 우클릭 → **Create** → **Material**
2. 새 머티리얼 선택 후 **Inspector**에서:
   - **Shader**: `Universal Render Pipeline` → **Unlit**
   - **Base Color**: 흰색 (255, 255, 255)
   - **Texture** (Base Map): 비워두거나 흰색 텍스처
3. 이 머티리얼을 무기 오브젝트의 **Mesh Renderer > Materials** 에 할당

## 3. 실루엣만 쓰고 본래 무기는 숨기기

- **실루엣용** 오브젝트: `PP_Theme_01_Axe_Two_Handed_002 (1)` 에 위 흰색 머티리얼 적용
- **실제 무기** 오브젝트: 기존 도끼(다른 오브젝트 또는 동일 계층의 다른 메시)는 평소엔 비활성화하고, 공격할 때만 활성화하는 식으로 쓰면 됩니다. (WeaponManager에서 나무/광석 무기 표시·숨김 처리와 같은 방식)

## 요약

| 목적           | 방법 |
|----------------|------|
| 하얀 실루엣     | URP **Unlit** 셰이더 + **Base Color 흰색** |
| 적용 위치      | 해당 무기 오브젝트의 **Mesh Renderer > Materials** |

이미지/아이콘용으로 “윤곽선만” 필요하면, 위처럼 Unlit + 흰색으로 두고 카메라/포지션으로 찍어서 사용하면 됩니다.
