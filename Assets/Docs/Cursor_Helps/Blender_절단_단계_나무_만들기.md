# Blender 절단 단계 나무 만들기 (순서대로)

Meshy에서 만든 나무 FBX를 Blender에서 **절단 단계별 모델(Stage1~4 + Stump)** 로 만드는 전체 순서입니다.

---

## 준비

- Meshy에서 내보낸 나무 **FBX 파일**이 있어야 합니다.
- Blender가 실행된 상태에서 시작합니다.

---

## 1단계: Blender에 나무 FBX 가져오기

1. **File → Import → FBX (.fbx)** 선택.
2. Meshy에서 받은 나무 `.fbx` 파일 선택 후 **Import FBX** 클릭.
3. 가져온 나무가 너무 작거나 크면 **Numpad . (점)** 으로 선택 오브젝트에 프레임 맞추기.
4. 필요하면 **S** (Scale)로 전체 크기 조절 후 **Enter**.

---

## 2단계: 메시 하나로 합치기 (필요할 때만)

**지금처럼 나무가 `Mesh1.0` 하나로만 보이면 → 이 단계는 건너뛰고 3단계로 가면 됩니다.**

Meshy 나무가 **여러 오브젝트**로 들어온 경우에만 합니다.  
(예: Outliner에 `Trunk`, `Foliage`, `Branch` 처럼 나무 부품이 2개 이상 있을 때)

1. **Outliner**에서 **나무 관련 메쉬만** **Shift+클릭**으로 전부 선택  
   (예: Mesh1.0, Mesh2.0, Leaf 등 — 나무를 이루는 모든 메쉬).
2. **Object → Join** (또는 **Ctrl+J**) 실행.
3. 하나의 오브젝트로 합쳐지면, 이름을 **Tree** 등으로 바꿔도 됩니다.

- **나무 메쉬가 1개뿐이면** 합칠 대상이 없으므로 **2단계는 하지 않습니다.**

---

## 3단계: 원본 나무 4개 복사 (Stage용)

절단할 나무를 4개 만들어 둡니다.

1. 나무 오브젝트만 선택한 상태에서 **Shift+D** (복제).
2. 마우스를 움직여 옆으로 띄운 뒤 **좌클릭**으로 위치 확정.
3. 같은 방식으로 **총 4번** 더 **Shift+D** 해서 나무가 **총 5개**가 되게 합니다.
   - 하나는 **원본(100%)**, 나머지 4개는 **75%, 50%, 25%, 0%(그루터기)** 용입니다.
4. Outliner에서 이름을 바꿔 두면 나중에 편합니다.
   - 예: `Tree_100`, `Tree_75`, `Tree_50`, `Tree_25`, `Tree_Stump`.

---

## 4단계: 첫 번째 복사본 — 75% 높이에서 자르기 (Tree_75)

1. **Tree_75**가 될 나무 하나만 선택.
2. **Tab** 키로 **Edit Mode** 진입.
3. **전체 선택**: **A** 키 두 번 (전체 선택).
4. **Bisect(자르기)** 실행:
   - 상단 메뉴 **Mesh → Bisect** (또는 **Ctrl+R** 후 Bisect 모드가 있으면 사용).
   - 또는 왼쪽 도구 패널에서 **Bisect** 도구 선택.
5. **자르는 선 그리기**:
   - 나무 **위쪽**에서 가로로 잘라야 하므로, **세로로 한 줄** 그어서 나무를 위/아래로 나눕니다.
   - 클릭 한 번 → 조금 위쪽에서 다시 클릭 (수직선 하나 생성).
6. **Bisect 옵션** (왼쪽 하단 **▷ Bisect** 펼치기):
   - **Fill** = **ON** (체크).  
     → 잘린 단면에 **폴리곤이 생겨서 구멍이 막힙니다.**
   - **Clear Outer** = **ON**.  
     → **자른 선 위쪽(바깥쪽)** 이 삭제되어, 위가 잘린 나무가 됩니다.
7. **Enter** 또는 **좌클릭**으로 자르기 확정.
8. **Tab**으로 **Object Mode**로 돌아옵니다.

이제 **Tree_75** = 위쪽이 잘리고, 위쪽 단면이 하나의 면으로 채워진 상태입니다.

---

## 5단계: 나머지 복사본도 같은 방식으로 자르기

- **Tree_50**: Tree_50 선택 → Edit Mode → Bisect로 **나무 높이의 절반** 정도에서 가로로 자르기 → **Fill ON**, **Clear Outer ON** → 위쪽 제거.
- **Tree_25**: Tree_25 선택 → **나무 높이의 1/4** 정도에서 같은 방식으로 자르기.
- **Tree_Stump**: Tree_Stump 선택 → **아주 낮게**(그루터기만 남기도록) 같은 방식으로 자르기.

매번:
- **Mesh → Bisect** 로 선 그리기  
- **Fill = ON**, **Clear Outer = ON**  
- 확정 후 **Tab** 으로 Object Mode 복귀  

---

## 6단계: 단면에 나이테 머티리얼 넣기 (나무 단면만 다른 재질)

잘린 **단면**만 나이테 재질로 바꾸려면, 단면 폴리곤에만 두 번째 머티리얼을 할당합니다.

### 6-1. 나이테 머티리얼 만들기

1. **Object Mode**에서 나무 하나 선택 (예: Tree_75).
2. 오른쪽 **Properties** → **Material Properties** (빨간 구 아이콘).
3. **+** 로 **새 슬롯 추가**.
4. **New** 로 머티리얼 생성.
5. 이름을 **WoodInside** 또는 **Ring** 등으로 변경.
6. **Base Color**를 밝은 갈색/나이테 색으로 설정 (나중에 Unity에서 텍스처로 바꿔도 됨).

### 6-2. 단면 면만 선택해서 재질 할당

1. 같은 나무 선택 상태에서 **Tab** → **Edit Mode**.
2. **Face** 선택 모드로 전환 (상단 **Face** 아이콘 또는 **3** 키).
3. 잘린 **단면**(평평한 원형/다각형 면)만 선택:
   - 단면 위에 커서 두고 **L** (Linked Select)로 연결된 면만 선택하거나,
   - **Alt+좌클릭**으로 엣지 루프 선택 후 단면만 남기거나,
   - **Ctrl+좌클릭**으로 단면 폴리곤만 직접 선택.
4. **Material Properties**에서 **WoodInside** 슬롯 선택 후 **Assign** 클릭.
5. **Tab**으로 Object Mode 복귀.

나머지 나무(Tree_50, Tree_25, Tree_Stump)도 같은 방식으로:
- **Material Properties**에서 **WoodInside** 머티리얼 추가(또는 기존 것 사용),
- Edit Mode에서 **단면 Face만 선택** → **Assign**.

---

## 7단계: FBX로 내보내기

각 스테이지를 **따로** FBX로 저장합니다.

1. **Object Mode**에서 **내보낼 나무 하나만** 선택 (나머지는 선택 해제).
2. **File → Export → FBX (.fbx)**.
3. 옵션 확인:
   - **Selected Objects** 체크 (선택한 것만 내보내기).
   - **Apply Modifiers** 체크.
   - **Apply Transform** 체크 권장.
   - **Forward**: -Z Forward (또는 프로젝트에 맞게).
   - **Up**: Y Up.
4. 파일명 예: `Tree_Stage1.fbx`, `Tree_Stage2.fbx`, … `Tree_Stump.fbx`.
5. **Export FBX** 클릭.
6. 다른 스테이지도 **선택 후 같은 방식**으로 각각 Export.

---

## 8단계: Unity에서 사용

1. 내보낸 FBX들을 Unity **Project** 창에 넣습니다.
2. 각 FBX에 머티리얼이 제대로 들어왔는지 확인.
3. **Bark**는 기존 나무 껍질, **WoodInside**는 단면(나이테)용 머티리얼로 두고,  
   Unity에서 **WoodInside**에 나이테 텍스처를 넣으면 됩니다.
4. 게임 로직에서 HP 구간별로:
   - HP 100~75 → Stage1  
   - HP 75~50 → Stage2  
   - HP 50~25 → Stage3  
   - HP 25~0 → Stage4  
   - 0 → Stump  
   처럼 프리팹을 바꿔 주면 됩니다.

---

## 요약 체크리스트

| 순서 | 할 일 |
|------|--------|
| 1 | FBX Import |
| 2 | (선택) 여러 오브젝트 Join |
| 3 | 나무 5개 복제 (100%, 75%, 50%, 25%, Stump) |
| 4 | 각 복사본 Edit Mode → Mesh → Bisect → **Fill ON**, **Clear Outer ON** |
| 5 | 단면 Face만 선택 → WoodInside 머티리얼 **Assign** |
| 6 | 스테이지별로 FBX Export → Unity에 넣기 |

**Bisect** 할 때마다 **Fill**을 켜야 단면이 막히고, **Clear Outer**를 켜야 잘린 위쪽이 지워져서 “도끼로 잘린 나무”처럼 보입니다.
