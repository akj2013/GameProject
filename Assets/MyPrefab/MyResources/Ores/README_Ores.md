# FarmLand 스타일 광석 (Ores)

- **메쉬**: Blender에서 Icosphere + Randomize + Z 0.7 → `OreRock.fbx` 하나만 사용.
- **텍스처**: `OreAtlas.png` (또는 `OreAtlas_Stone_Gold_Ruby_Sapphire.png`) — Stone / Gold / Ruby / Sapphire 2×2 아틀라스.
- **머티리얼**: Tiling (0.5, 0.5) + Offset으로 종류별로 한 칸만 쓰기 (MAT_Ore_Stone / Gold / Ruby / Sapphire).
- **프리팹**: OreStone, OreGold, OreRuby, OreSapphire (메쉬 동일, 머티리얼만 교체).

자세한 순서는 **Assets/Docs/Blender_광석_UV부터_Unity_적용.md** 참고.
