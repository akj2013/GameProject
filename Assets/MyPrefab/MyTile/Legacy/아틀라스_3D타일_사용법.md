# 아틀라스 텍스처를 3D 타일 머티리얼에 쓰는 방법

## 왜 Sprite(TEX_Tile_Set_0)를 Base Map에 넣을 수 없나?

- **Base Map**은 **Texture2D**만 받습니다.
- Sprite Editor로 잘린 `TEX_Tile_Set_0`, `TEX_Tile_Set_1` … 은 **Sprite** 타입이라 3D Lit 머티리얼의 Base Map에 드래그해도 적용되지 않습니다.
- 따라서 **아틀라스 전체 텍스처 한 장**을 Base Map에 넣고, **Tiling / Offset**으로 “한 칸”만 보이게 하는 방식으로 씁니다.

---

## 올바른 설정 순서

### 1. Base Map에는 **아틀라스 본체**만 넣기

- Project에서 **TEX_Tile_Set** (png 아이콘, 스프라이트가 아닌 텍스처 에셋)을 선택합니다.
- 머티리얼 Inspector → **Base Map**에 이 **TEX_Tile_Set**을 드래그합니다.  
  (TEX_Tile_Set_0, _1 같은 **스프라이트**는 넣지 않습니다.)

### 2. Tiling = 한 칸 크기

현재 아틀라스는 **4열 × 2행**으로 잘려 있습니다.

- **Tiling X** = 1/4 = **0.25**
- **Tiling Y** = 1/2 = **0.5**

(2×3이었다면 Tiling = (0.5, 0.333) 이 됩니다.)

### 3. Offset = 타일 번호별 위치

UV에서 (0,0)이 왼쪽 아래입니다.  
각 타일은 **왼쪽 아래 모서리** 기준으로 다음처럼 잡으면 됩니다.

| 머티리얼 예시       | 타일 | Offset X | Offset Y |
|--------------------|------|----------|----------|
| MAT_Tile_Grass     | _0   | 0        | 0.5      |
| (두 번째 타일)     | _1   | 0.25     | 0.5      |
| (세 번째 타일)     | _2   | 0.5      | 0.5      |
| (네 번째 타일)     | _3   | 0.75     | 0.5      |
| (다섯 번째 타일)   | _4   | 0        | 0        |
| (여섯 번째 타일)   | _5   | 0.25     | 0        |
| (일곱 번째 타일)   | _6   | 0.5      | 0        |
| (여덟 번째 타일)   | _7   | 0.75     | 0        |

- **Tiling**은 위와 같이 **모든 타일 머티리얼 공통**: **(0.25, 0.5)**  
- **Offset**만 타일마다 위 표처럼 다르게 넣으면 됩니다.

---

## Unity에서 직접 넣는 방법

1. 머티리얼 선택 → Inspector에서 **Base Map** 옆 작은 회색 네모(텍스처 슬롯) 클릭.
2. 검색창에 `TEX_Tile_Set` 입력 후 **텍스처 에셋**(스프라이트가 아닌 것) 선택.
3. **Tiling**: X = 0.25, Y = 0.5
4. **Offset**: 위 표에서 해당 타일 번호에 맞는 X, Y 입력.

이렇게 하면 **한 머티리얼당 아틀라스의 한 칸**만 보이게 됩니다.

---

## 요약

- Base Map에는 **TEX_Tile_Set (텍스처)** 만 넣기. **TEX_Tile_Set_0** 같은 스프라이트는 넣지 않기.
- 4×2 아틀라스: **Tiling (0.25, 0.5)** 고정, **Offset**만 타일마다 다르게 설정.
